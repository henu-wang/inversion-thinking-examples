# Inversion Thinking Examples

Charlie Munger's favorite mental model: instead of thinking about how to succeed, think about how to fail, then avoid those things. This collection provides real-world examples.

## Overview

This project is part of the [KeepRule](https://keeprule.com) decision-making knowledge ecosystem. It provides practical tools and frameworks for improving your thinking and decision quality.

## Key Features

- Comprehensive framework documentation
- Real-world examples and case studies
- Practical exercises and worksheets
- Integration with other mental models
